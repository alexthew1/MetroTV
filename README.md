# MetroTV

MetroTV is a high-performance, cross-platform IPTV application built with Electron. Designed with a modern Metro-inspired aesthetic, it provides a seamless experience for managing multiple IPTV playlists with full EPG integration and fluid navigation.

## Features

* **Multi-Playlist Support:** Manage and switch between multiple IPTV sources effortlessly.
* **Metro UI Design:** Responsive, hardware-accelerated interface with customizable accent colors.
* **Advanced EPG:** Full Electronic Program Guide support with live progress bars and detailed program metadata.
* **Fluid Navigation:** Full-screen channel switching and smooth loading transitions.
* **Cross-Platform:** Native-feel performance on Windows and macOS. (Linux is in the works)

## Getting Started

1.  **Download:** Grab the latest installer for your OS from the [Releases](https://github.com/alexthew1/MetroTV/releases) tab.
2.  **Install:** Run the executable and follow the prompts.
3.  **Configure:** Add your M3U playlist URL and XMLTV EPG link in the settings tab to begin watching.
